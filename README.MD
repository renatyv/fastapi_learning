# FastAPI blog API
Implements a blog backend API.

# Run in development mode
```$ uvicorn blog.main:app --reload```
Server is loaded at  
http://127.0.0.1:8000

# Example
try in browser  
* ```http://127.0.0.1:8000/posts```  
* ```http://127.0.0.1:8000/users```
# API docs
```http://127.0.0.1:8000/docs```

## Dependencies
* Python 3.9
* FastAPI
